
인라인요소 안에는 블럭요소를 넣을 수 없다
블럭요소 안에 인라인 가능

span - 인라인요소
콘텐츠 영역을 설정하는 용도
콘텐츠 크기만큼자동으로 줄어듬
인라인 글자요소는 가로세로 사이즈를 지정할 수 없다

div(division) 블럭요소 수직으로 쌓임
가로넓이는 최대한 늘어나려고 한다
가로세로 사이즈를 지정할 수 있다
마진 외부 /  패딩 내부 다 가능

h1 (헤딩) 블록
p (문장) 블록
img (이미지) 인라인
ul (unordered list 순서필요없는 목록집합) 블록
li (list Item) 블록
<ul과 li 꼭 세트>
a (anchor 하이퍼링크 지정) 인라인
br(break 줄바꿈) 인라인
input (데이터입력. 사용자가 데이터입력) 인라인블럭
<수평으로 쌓임+가로세로 지정가능>
label (input의 제목) 인라인

table (표 행 열의 집합) 테이블
<행이먼저 그다음 열
tr (table row 행)
td (table data 칸, 셀)

[ 일반적인 CSS 우선순위! ]
 !important > inline > id > class > type
